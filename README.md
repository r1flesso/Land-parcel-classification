# Land-parcel-classification
ML-проект на Python для классификации типов земель (на основе реальных данных) с использованием scikit-learn, imbalanced-learn (SMOTE, RandomOverSampler, RandomUnderSampler), optuna, MLflow. Сравниваются разные модели машинного обучения, а также подходы, решается проблема дисбаланса классов

## Установка
1. Клонируйте мой репозиторий:
    ```bash
    git clone https://github.com/r1flesso/Land-parcel-classification
    ```
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```
3. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```
